# Task Template: Bug Fix

## Workflow
```
Debug → Coder → Tester → Reviewer → Report
```

## Steps
1. **Debug**: Reproduce, isolate, analyze root cause
2. **Coder**: Implement the fix
3. **Tester**: Write regression test, verify fix
4. **Reviewer**: Review fix quality
5. **Orchestrator**: Generate report

## Acceptance Criteria
- [ ] Bug is fixed
- [ ] Root cause is documented
- [ ] Regression test added
- [ ] All existing tests still pass
- [ ] No new issues introduced
