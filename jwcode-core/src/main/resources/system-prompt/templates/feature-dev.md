# Task Template: Feature Development

## Workflow
```
Explore → Architect → Coder → Tester → Reviewer → Report
```

## Steps
1. **Explore**: Analyze existing codebase structure and relevant modules
2. **Architect**: Design the feature interface and data models
3. **Coder**: Implement the feature
4. **Tester**: Write and execute tests
5. **Reviewer**: Review code quality
6. **Orchestrator**: Generate final report

## Acceptance Criteria
- [ ] Feature works as specified
- [ ] All tests pass
- [ ] Code review passes (no critical issues)
- [ ] Documentation updated
- [ ] Backward compatible (unless intentionally breaking)
